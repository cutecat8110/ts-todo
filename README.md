# TS Todo

> 這是一個 Vue3 + TypeScript 的練習專案，提供註冊功能的待辦事項應用程式。

<br />

## 🌸 安裝與使用

### 系統需求

建議 Node.js 20.11.1 或更高版本。

### 取得專案

```sh
git clone https://github.com/cutecat8110/ts-todo.git
```

### 設定環境

複製 .env.example 改為 .env

```sh
cp .env.example .env
```

### 安裝依賴

```sh
npm install
```

### 啟動開發環境

```sh
npm run dev
```

<br />
